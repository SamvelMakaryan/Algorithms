# NxMQueens solver

# Introduction

This is a C++ program that solves the N*M Queens problem.The N*M Queens problem is a classic chessboard problem where you need to place N queens on an NÃ—M chessboard in such a way that no two queens can attack each other.

## Features

- Finds and prints all distinct solutions for the N*M Queens problem.
- Supports any board size by specifying N and M.
